    Settings

    Generation

    Leave a field blank to use the model/provider default. Thinking models ignore top_p/top_k and force temperature; Anthropic does not support the penalty parameters.

    Display
    Planning

    Turn on planning for a single request with the Plan toggle next to the message box (or by sending "/plan <task>"); a planned turn drafts a plan first. These options shape a planned turn: "By you" pauses for your Approve / Edit / Reject; "by an agent" uses an independent reviewer (no human pause). With both plan reviews on, the agent's critique is shown to you before you decide.
